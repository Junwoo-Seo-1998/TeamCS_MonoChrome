Installation instructions:
You need to double click MonoChrome_Setup.exe for install monochrome at your computer.

¡°How To Play¡± instructions: 
You need approach destination with platform moving and rotating.

Also, you can change platform's color by absorb paint and shoot it to platform except black color platoform.

You can enter platform control mode by click mouse middle button when you are not stand on black
platform.

When you enter platform control mode, you cannot move.
Also, if color platform is collide with player when platform control mode,
platform control mode is change to player control mode by forced.

Controls:
-When player control mode(default)
	-W to Jump
	-A, D to move left, right 
	-When player absorb paint and Gun's color is changed	
		-click mouse left button for shot absorbed color
	-mouse middle button click for change to platform control mode

-When platform control mode (Arrow point which platform you can control)
	-click mouse left button for select left platform
	-click mouse right button for select right platform 
	-If selected platform's color is blue
		-W A S D to move platform Up Down Left Right
	-If selected platform's color is green
		-Q E to rotate platform counter clockwise or clockwise 
	-mouse middle button click for chnage to player control mode 

Cheat codes:
If you push C button, you can skip current level.
